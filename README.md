# Vyntaq AI 🚀

AI-powered data analysis platform that converts raw datasets into insights, visualizations, and business recommendations.

## Features
- 📂 Upload CSV datasets
- 📊 Interactive visualizations (Plotly)
- 🧠 AI-generated insights (LLM integration)
- 🔍 Data validation and cleaning

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- OpenRouter API

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
